# 🎬 CineMatch

Aplikasi web pemesanan dan rekomendasi film berbasis **Laravel API** sebagai backend dan **React Vite** sebagai frontend.

## 🚀 Teknologi yang Digunakan

### Backend
- Laravel
- PHP
- MySQL
- Laravel API
- Composer

### Frontend
- React
- Vite
- Tailwind CSS 

### 📁 Struktur Project
````
CineMatch
│
├── backend
│   ├── app
│   ├── routes
│   ├── database
│   └── composer.json
│
└── fe-cinema
    ├── src
    ├── public
    ├── package.json
    └── vite.config.js
````
